# Hadia Mushtaq | MERN Stack Developer Portfolio

Welcome to my personal portfolio website, built with **React** and **React Router**, showcasing my projects, skills, and contact information. This portfolio is fully **responsive**, clean, and professional, designed to give recruiters and potential employers a clear view of my work.

---

## Live Demo

[View Live Portfolio](https://portfolio-1-0-theta.vercel.app/)

---

## Tech Stack

- **Frontend:** React, HTML5, CSS3, JavaScript, React Icons
- **Routing:** React Router
- **Design & Animations:** CSS Grid, Flexbox, hover effects, subtle scroll animations
- **Responsive Design:** Mobile-first layout, tablet and desktop optimized

---

## Features

- **Home Section:** Professional introduction with a call-to-action button linking to projects.
- **Skills Section:** Interactive tech stack showcasing my main tools and technologies.
- **About Section:** Professional summary detailing my journey, skills, and projects.
- **Projects Section:** Cards displaying projects with descriptions and live links:
  - **Boggo:** Blog app with CRUD operations, authentication, and search.
  - **Sticky:** Notes app with CRUD operations and authentication.
  - **JWT:** Authentication app with login/logout functionality.
  - **Shoppily (React):** E-commerce website using API integration.
  - **Jobster (React):** Job search and posting platform with dual UI for users and recruiters.
- **Contact Section:** Email, LinkedIn, GitHub, and phone links with icons.
- **Navbar:** Responsive navigation with hamburger menu on small screens.
- **Footer:** Copyright notice with current year.
